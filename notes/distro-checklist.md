# Distro Checklist

## Pendientes

- [ ] **Debian Server** — estable, base de servidores
- [ ] **Ubuntu Server** — popular, compatibilidad asegurada
- [ ] **Alpine Linux** — minimal, contenedores, musl + openrc
- [ ] **Void Linux** — rolling, runit, lightweight
- [ ] **Gentoo** — source-based, aprendizaje profundo
- [ ] **NixOS** — inmutable, declarativo, paradigmático
- [ ] **Slackware** — la más antigua activa, simplicidad UNIX
- [ ] **FreeBSD** — no es Linux, pero sí UNIX-like
- [ ] **openSUSE Tumbleweed** — rolling estable
- [ ] **Fedora Server** — innovación, SELinux por defecto
- [ ] **Rocky Linux** — clon de RHEL, enterprise
- [ ] **Artix Linux** — Arch sin systemd (openrc/runit/s6)
- [ ] **Kali Linux** — pentesting, hardening
- [ ] **Tails** — anonimato, amnesic, tor
- [ ] **Qubes OS** — virtualización por defecto, security-focused

## Criterios por probar

Cada distro debería documentar:

- [ ] Instalación manual paso a paso por terminal
- [ ] Post-instalación (usuario, sudo, ssh, herramientas)
- [ ] Errores encontrados con solución
- [ ] Funciona como VM headless (consola serie)
- [ ] Speedrun posible? (sí/no y tiempo)
- [ ] Variante servidor
- [ ] Variante "difícil" (hardened, minimal, personalizada)
